Installing Visual Studio

If you have already have a favourite C/C++ IDE then please feel free to skip this step. I will be using Visual Studio throughout this tutorial

To install Visual Studio, first download the installer from here.Clicking the link takes you to website which should look like this:

Hover over the “Download Visual Studio” button and select “ Community 20XX” (with the XX standing for the current year e.g Community 2019). This will automatically begin downloading the installer. Wait for it to download and then run the installer. Follow the steps of the installer. Once the installer has completed installation, run Visual Studio to check if the installation was successful.

We won’t need to use Visual Studio in the next section, so you can close the application down for now.
