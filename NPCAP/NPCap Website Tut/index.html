Welcome to my NPCap tutorial

Hello! In this tutorial we will be learning how to use NPCap, from install to sniffing packets.

First of all, what is NPCap?

Npcap is a packet sniffing (and sending) library/architecture for Windows. It is based on the discontinued WinPcap library, but with improved speed, portability, security, and efficiency. It is called sometimes refrenced as an architecture as, opposed to a standard programming library, NPCap interacts directly with your computer's networking hardware.
