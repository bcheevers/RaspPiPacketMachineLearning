Installing NPCap SDK

In this section we learn how to install the NPCap SDK.

Preamble

The NPCap SDK (Software Development Kit) will contain the code files we will reference in our own code. Inside the SDK file itself will be the header (.h) files and the library (.lib) files for the NPCap library/architecture.

Header files are files that tell the compiler how to call some functionality (without knowing how the functionality works) are called header files. They contain the function prototypes. They also contain Data types and constants used with the libraries. We use #include to use these header files in programs. This would be similar import in other programming languages.

Library files are files that contain the actual functionality i.e. they contain function body.
